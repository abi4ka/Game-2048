# 2048 Game (JavaFX)

A clean and minimal implementation of the classic **2048** game written in **Java** using **JavaFX**.  
The project focuses on clear game logic, simple UI rendering, and keyboard-based controls.

---

## Features

- Classic 4Ã—4 2048 gameplay
- Keyboard controls (`W`, `A`, `S`, `D`)
- Win and lose condition detection
- Dynamic tile rendering with colors
- Separation of game logic and UI

---

## Controls

| Key | Action |
|----|--------|
| W  | Move Up |
| A  | Move Left |
| S  | Move Down |
| D  | Move Right |

---

## Technologies Used

- **Java**
- **JavaFX**

---

## Dependencies

This project uses **JavaFX** as the UI framework.

### JavaFX (via Gradle)
- `javafx-controls`
- `javafx-graphics`

Recommended JavaFX version: **21**

---

## Build & Run

The project is configured to run via **Gradle**.

```bash
./gradlew run
````
